Backend Installation

We need to create the "Brain" of the system using Google Apps Script.

  1. Open Google Apps Script in a new tab.
  2. Click New Project.
  3. Click Project Settings (Gear Icon). Check "Show 'appsscript.json' manifest file in editor".
  4. Copy the code below and paste it into the Code.gs file (delete existing code first).

  • Click Deploy > New Deployment.
  • Select type: Web App.
  • Execute as: Me.
  • Who has access: Anyone (Crucial for the App to work).
  • Click Deploy and Copy the Web App URL.

Database Setup

Now we configure the Google Sheet that stores your data.

  1. Run your new Web App URL once in your browser. It should say {"status":"online"}.
  2. This automatically created a Google Sheet in your Drive named "New Sheet". Find it and rename it to OTG_Database.
  3. Open the Sheet. We need to set up the headers.
